O que é Semântica

Semântica significar "dar significado". Aqui no html5 a semântica é o principal motivo de ele ter atualizado e
melhorado porque no html4 tinha algumas tags que alteravam a forma do site e isso não se deve fazer mais, isso foi substituido para o css

Forma e significado

Agora no html5 não é mais coerente usar tags que alterem a forma do site, por exemplo cor do fundo da página, alterar cor de palavras e etc.
