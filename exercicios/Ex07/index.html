Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b>. Porém ela não é semântica.
Logo podemos usar a tag <strong> porque ela é semântica por conta do significado da tag, que quer dizer uma palavra forte, uma palavra em destaque.

Itálico / Ênfase

Nesta frase, temos um termo em Itálico usando a tag <i>, porém ela não é semântica.
Nesta outra faze temos um termo em ênfase usando a tam <em>, mas esta é uma tag semântica.

Texto Marcado

Podemos criar um texto marcado usando a tag <mark>.

Texto "excluído"

Podemos dizer que um texto foi excluído colocando a tag <del>

Texto Sublinhado

Para sublinhar um texto colocamos a tag <ins>.
Temos também a tag <u> para sublinhar um texto, porém ela não é semântica.

Texto Sobrescrito

Podemos sobreescrever um texto, por exemplo indicar uma raíz quadrada, x2+1

Texto Subescrito

Para subescrever um texto utilizamos a tag <sub>, por exemplo: H2O
