Parágrafos e quebra de linha

Parágrafos se fazem com a tag <p>, é simplesmente digitar p no editor de código, pelo menos no visual estudio code.

Para fazer uma quebra de linha como essa,
basta digitar uma tag chamada <br>, ela significa brake roll que traduzindo é "quebra de linha".

Simbolos especificos

Existem algumas tags que são simbolos especiais:
®
©
™
Aqui estão também algumas marcas para dinheiro:
€ ← euro
$ ← Dollar
ℜ ← Real
£ ← libra
¥ ← Moeda japonesa
¢ ← Centavo americano
