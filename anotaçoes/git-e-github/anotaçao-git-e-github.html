O que é git e github

Ao longo dos projetos que temos, sites em criação, aplicativos e etc.. Mudamos constantemente os arquivos e a forma como ele está, o projeto vai mudando bastante ao longo do tempo e com isso precisamos de algo para guardar as mudanças. Por este motivo foram criadas o git e o github.

Git

O git a primeira vista serve para salvar versões anteriores de algum projeto em desenvolvimento. Ele salva os arquivos de forma local, na própria máquina, ou seja, no hd ou ssd. Chamamos isso de repositório local

Github

O github tem a mesma funcionalidade do git, porém a forma de salvamento dos arquivos é feita de forma remota, na nuvem. Chamamos isso de repositório remoto.

Por que usar

Os arquivos devem sempre ser salvos e esses softwares ajudam bastante nesse ponto, seria complicado e nada prático salvar cada alteração feita nos computadores, além de muitos arquivos que iriam ser criados, também confundiria muito a cabeça de quem está salvando, algum dia poderia esquecer de fazer e daria muito trabalho para refazer ou desfazer algo que fez.
