O que é Media Queries

O conceito de media queries foi criado porque nasceu a necessidade dos sites poderem ser acessados por diferentes tamanho de dispositivos, por exemplo, acessar o youtube em um monitor de 25 polegadas é totalmente diferente de acessar ele em um celular. Logo o CSS foi adaptando e criando funções que possibilite esses diferentes tipos de tela.

Media Queries estão nos estilos diferentes do CSS, para imprimir uma página por exemplo, não é necessário mostrar diversos conteúdos, como links, propagandas, cores e etc.. Logo, é necessário criar outro arquivo CSS e adapta-lo para o modo de impressora. Isto é Media Queries
